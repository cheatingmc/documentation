## API Questions

<AccordionGroup>
  <Accordion title="What are the rate limits?">
    You can send up to 100 requests per minute.
  </Accordion>
  <Accordion title="How do I check stock before purchasing?">
    Use the `/api/stock` endpoint to check available inventory before making a purchase. This ensures you know what's available.
  </Accordion>
  <Accordion title="What happens if stock runs out?">
    Our system automatically restocks accounts. You can enable notifications in Discord to be alerted when stock is replenished.
  </Accordion>
</AccordionGroup>
