## Getting your API Key

<Steps>
  <Step title="Join Discord">
    Join our Discord server to get started.
  </Step>
  <Step title="Get your Key">
    Use the command `/api-key` in the appropriate channel.
  </Step>
</Steps>

<Warning>
  Never share your API key publicly. Anyone with your key can make purchases on your behalf.
</Warning>

<br />

## Using Your API Key

Include your API key in the request url:

```bash
curl -X GET "https://api.nicealts.com/v1/stock?key=YOUR_API_KEY"
```

<CardGroup cols={2}>
  ## Error Responses
</CardGroup>

If authentication fails, you'll receive:

```json
{"detail": "Invalid API key"}
```
