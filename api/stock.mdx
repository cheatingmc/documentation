# Stock Endpoint

This endpoint returns current stock levels for all account types.

## Endpoint

```
GET https://api.nicealts.com/v1/stock
```

## Request

<CodeGroup>

```bash cURL
curl -X GET "https://api.nicealts.com/v1/stock?key=YOUR_API_KEY"
```

</CodeGroup>
