# Stock Endpoint

Check available inventory before making purchases. This endpoint returns current stock levels for all account types.

## Endpoint

```
GET https://api.nicealts.com/stock
```

## Request

<CodeGroup>

```bash cURL
curl -X GET "https://api.nicealts.com/stock" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```python Python
import requests

headers = {
    "Authorization": "Bearer YOUR_API_KEY"
}

response = requests.get("https://api.nicealts.com/stock", headers=headers)
stock = response.json()
print(stock)
```

```javascript JavaScript
const response = await fetch('https://api.nicealts.com/stock', {
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY'
  }
});

const stock = await response.json();
console.log(stock);
```

</CodeGroup>

## Response Fields

<ResponseField name="success" type="boolean">
  Indicates if the request was successful
</ResponseField>

<ResponseField name="stock" type="object">
  Object containing stock information for each account type
  
  <Expandable title="Account Types">
    <ResponseField name="unbanned_accesstoken" type="object">
      Full access unbanned accounts with AccessToken
      - `available`: Number of accounts in stock
      - `price_credits`: Price per account in credits
    </ResponseField>
    
    <ResponseField name="unbanned_cookie" type="object">
      Unbanned accounts with Cookie authentication
    </ResponseField>
    
    <ResponseField name="unbanned_emailpass" type="object">
      Unbanned accounts with email:password format
    </ResponseField>
    
    <ResponseField name="banned_accesstoken" type="object">
      Banned accounts with AccessToken (discounted)
    </ResponseField>
    
    <ResponseField name="banned_cookie" type="object">
      Banned accounts with Cookie authentication (discounted)
    </ResponseField>
    
    <ResponseField name="banned_emailpass" type="object">
      Banned accounts with email:password format (discounted)
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="last_updated" type="string">
  ISO 8601 timestamp of when stock was last updated
</ResponseField>

## Example Use Case

Before purchasing, check if your desired account type is in stock:

```python
import requests

def check_stock(api_key, account_type):
    headers = {"Authorization": f"Bearer {api_key}"}
    response = requests.get("https://api.nicealts.com/stock", headers=headers)
    
    if response.status_code == 200:
        stock = response.json()
        if stock['stock'][account_type]['available'] > 0:
            print(f"✓ {account_type} available: {stock['stock'][account_type]['available']}")
            return True
        else:
            print(f"✗ {account_type} out of stock")
            return False
    else:
        print("Error checking stock")
        return False

# Check before purchase
if check_stock("your_api_key", "unbanned_accesstoken"):
    # Proceed with purchase
    pass
```
